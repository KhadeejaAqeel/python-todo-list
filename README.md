# Python To-Do List

A simple command-line To-Do List application built using Python. This project helps users manage their daily tasks through a menu-driven interface.

## Features

- Add new tasks
- View all tasks
- Delete completed tasks
- Simple command-line interface

## Technologies Used

- Python 3

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Run the following command:

```bash
python todo.py
